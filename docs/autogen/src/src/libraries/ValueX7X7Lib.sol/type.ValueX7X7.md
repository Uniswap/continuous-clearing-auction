# ValueX7X7
[Git Source](https://github.com/Uniswap/twap-auction/blob/57168f679cba2e43cc601572a1c8354914505aab/src/libraries/ValueX7X7Lib.sol)

A ValueX7X7 is a ValueX7 value that has been multiplied by MPS

*X7X7 values are used for supply values to avoid intermediate division by MPS*


```solidity
type ValueX7X7 is uint256;
```
